Projects

Question Answering System using Natural Language Processing

Developed a novel pipeline for solving multiple choice question which leverages various NLP techniques such as coreference resolution, information retrieval in the form of knowledge graphs, knowledge graph embeddings and a unique answer selection module

Pneumonia Detection and Classification using Computer Vision

Developed a novel pipeline for delection and classification of Pneumonia using lung radiographs data and employing super-pixel segmentation and colorization techniques. Ensemble of modified ViT and YOLOv5 models was used.

Fashion Trend Detection and Prediction

Developed a model that used web-scrapping for collection of image dataset from instagram, employed MaskRCNN, InceptionV3 and LSTM models for segmentation and attribute detection of clothes from scraped images and applied data analytics to observe trends for each attribute of clothes.

NYC Airbnb Market Analysis

analyzed the NYC Airbnb market by combining data from multiple datasets to compare Airbnb prices to private market prices and compare Airbnb prices across 5 NYC boroughs. Used pandas and Numpy for entire analysis.

A/B Test for Videogame App

Analyzed an AB-test where the first gate in the game is moved from level 5 to level 10. Derived the impact of gate-shift on player retention, considering 1-day and 7-day retentions.

Keyword Generation for Product Advertisement

Developed a prototype for generating an exhaustive and effective list of Search Engine Keywords. It can target any specific product, and can be useful for GoogleAds.
